# 快速使用指南

> 💡 3 分钟快速上手，开始备份您的重要聊天记录

## 如何使用 AImyChats

AImyChats 通过简单的邮件转发，让您轻松备份微信、邮件等重要聊天记录。

## 快速开始

### 1. 注册账号

访问 AImyChats 平台（aimychats.com），创建您的专属账号。注册后，您将获得一个专属的邮箱地址。

### 2. 获取专属邮箱

登录后，在设置页面找到您的专属备份邮箱地址，格式类似：

```
your-unique-id@aimychats.com
```

### 3. 在微信中转发

当您需要备份某个重要的微信对话时：

1. 长按微信聊天记录
2. 选择"转发"或"分享"
3. 选择"通过邮件发送"
4. 收件人填写您的 AImyChats 专属邮箱
5. 发送即可

## 功能说明

### AI 图片识别 + 多场景智能分析

AImyChats 的核心优势 - 图文结合 + 场景化智能分析：

**图片识别**：
- **自动识别图片内容**：截图、图表、照片中的关键信息
- **OCR 文字提取**：图片中的文字自动识别
- **图表数据理解**：数据图、统计图自动分析

**多场景智能分析**：
- **选择场景**：工作沟通、客户咨询、产品讨论、技术支持等
- **针对性分析**：不同场景侧重点不同，更精准
- **客服场景**：重点提取客户问题和诉求
- **产品场景**：重点识别功能需求和细节
- **工作场景**：重点提取任务和责任人

**图文 + 场景结合**：
- **对话主题识别**：结合文字、图片、场景，准确识别对话核心
- **关键信息提取**：从文字和图片中提取场景相关的重要信息
- **更准确的摘要**：因为 AI 真的"看懂"了图片内容，并理解场景需求！

### 图片永久保存

微信图片通常会在一段时间后过期，但通过 AImyChats：

- 所有图片自动上传到云端存储
- 永久可访问，不会过期
- 支持原图质量保存
- **图片内容已被 AI 识别和索引，搜索更方便**

### 智能搜索

使用强大的搜索功能快速找到历史记录：

- **关键词搜索**：根据内容关键词查找
- **时间筛选**：按日期范围筛选
- **AI 分类**：自动分类为工作、生活等不同场景

### Q: 图片保存有数量限制吗？
