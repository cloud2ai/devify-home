## 关键更新点

### 1. 存储空间大幅提升
- Free: 10MB → **1 GB** (100x 提升)
- Starter: 50MB → **5 GB** (100x 提升)
- Standard: 200MB → **10 GB** (50x 提升)
- Pro: 1000MB → **20 GB** (20x 提升)

### 2. 数据保留期延长
- Free: 30天（不变）
- Starter: 90天 → **1年** (4x 延长)
- Standard: 180天 → **3年** (6x 延长)
- Pro: 365天 → **永久** (∞)

### 3. 附件限制更清晰
- 从"附件大小限制"改为"每封邮件附件数量限制"
- Free: 5个/封
- Starter: 10个/封
- Standard: 15个/封
- Pro: 30个/封

### 4. 邮件处理量调整
- Free: 5封/月（不变）
- Starter: 10封/月 → **100封/月**（大幅提升）
- Standard: 20封/月 → **500封/月**（大幅提升）
- Pro: 30封/月 → **2000封/月**（聚焦高量级用户）

### 5. 单价透明化
- Starter: $0.0499/封（极具竞争力）
- Standard: $0.0198/封（最高性价比）
- Pro: $0.0150/封（量大更划算）

### 6. 功能分配优化
- 数据导出：Professional+ → **仅 Pro**
- API 访问：Professional+ → **仅 Pro**
- SSO 集成：Enterprise → **仅 Pro**
- CRM 集成：Enterprise → **仅 Pro**

### 7. 支持响应时间
- Free/Starter: 48小时
- Standard: 24小时
- Pro: 4小时（专属支持）

### 8. API 速率限制
- Free: 100/小时
- Starter: 500/小时
- Standard: 2,000/小时
- Pro: 10,000/小时
